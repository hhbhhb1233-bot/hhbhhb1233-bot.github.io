iPhone / iPad | 系统版本 iOS 17, 18, 26

一键部署

复制命令到终端执行, 自动完成下载和安装

前置要求

  • iPhone / iPad, 系统版本 >= iOS 17
  • 一台电脑 + 一根数据线
  • 已安装 Python 3.12+ (安装时勾选 Add to PATH)
  • Windows 用户需安装 iTunes (提供 iPhone USB 驱动, 命令会自动安装)

必须以管理员身份打开 PowerShell (右键 -> 以管理员身份运行)

# 安装 iTunes + uv + 克隆项目 + 安装依赖 + 运行
winget install --id Apple.iTunes --accept-source-agreements --accept-package-agreements && powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex" && git clone https://github.com/BiancoChiu/iOSRealRun-cli-18.git && cd iOSRealRun-cli-18 && uv sync && uv run main.py
# 安装 uv + 克隆项目 + 安装依赖 + 运行
curl -LsSf https://astral.sh/uv/install.sh | sh && git clone https://github.com/BiancoChiu/iOSRealRun-cli-18.git && cd iOSRealRun-cli-18 && uv sync && sudo uv run main.py

首次运行会自动下载依赖, 之后再次运行只需复制下方命令:

cd iOSRealRun-cli-18 && uv run main.py
2

使用说明

连接设备, 开始模拟跑步

操作流程

  1. 用数据线将 iPhone/iPad 连接电脑, 点击信任此电脑
  2. 运行上方命令后, 程序会自动开始模拟跑步
  3. 项目目录中已内置路线 (HNroute.txt), 可直接使用
  4. 按 Ctrl+C 停止模拟

注意事项

  • 请勿直接关闭窗口, 否则 GPS 定位无法恢复 (重启手机可修复)
  • Windows 提示"无设备连接"时, 请确认已安装 iTunes
  • 确保只连接一台 iOS 设备

自定义路线 (可选)

如需修改路线, 可导出路线文件替换项目中的 HNroute.txt 和 config.yaml。

Android 手机虚拟跑步

安卓真机 | 系统版本 Android 11+

1

下载安装

下载 FakeGPS 并安装到手机

前置条件

  • Android 手机, 系统版本 Android 11 及以上
  • 无需 Root

下载 APK

下载 FakeGPS.apk (5.4MB)

在手机浏览器中打开本页下载, 或电脑下载后传到手机安装

真机上不支持 Keep。如需在 Keep 上使用, 请返回首页选择电脑模拟器方案。

2

设备配置

开启开发者选项并设置模拟位置

开启开发者选项

  1. 打开 设置 > 关于手机
  2. 连续点击 "版本号" 7 次, 直到提示"已进入开发者模式"
  3. 返回设置, 找到 开发者选项 (部分手机在"系统"或"更多设置"中)

设置模拟位置应用

  1. 在 开发者选项 中找到 "选择模拟位置信息应用"
  2. 选择 FakeGPS

推荐设置

  • 使用 悬浮窗 方式前台运行, 后台运行可能不稳定
  • 关闭系统的省电优化 / 后台限制 (针对 FakeGPS)
3

4

运行模拟

启动 FakeGPS 开始虚拟跑步

  1. 打开 FakeGPS, 授予定位权限
  2. 调整速度 (建议 2.5 - 4.0 m/s)、圈数
  3. 点击 "开始"
  4. 切换到运动 App (小米运动健康、华为运动健康等), 开始记录

注意事项

  • 推荐使用悬浮窗前台运行, 后台可能中断
  • 如定位未恢复, 关闭 FakeGPS 或重启手机即可
  • 速度不宜超过 5 m/s, 否则可能被判定异常
  • 真机不支持 Keep, 支持小米运动健康、华为运动健康

电脑模拟器虚拟跑步

无需手机 | 支持 Keep | 稳定性更好

1

安装模拟器

下载并安装 Android 模拟器

推荐模拟器 (任选其一)

  • BlueStacks 5 (推荐, 选择 Android 11 系统)
  • 雷电模拟器 (选择 Android 11 版本)
  • MuMu 模拟器 12

模拟器无真实 GPS 硬件, 不会出现定位冲突, 稳定性比真机更好。且模拟器上支持 Keep、小米运动健康等所有主流运动 App。

下载 FakeGPS APK

下载 FakeGPS.apk (5.4MB)

安装 APK 到模拟器

  • 方式一: 直接将 APK 文件拖入模拟器窗口
  • 方式二: 使用模拟器自带的文件管理器安装
  • 方式三: adb install FakeGPS.apk

运行模拟

在模拟器中启动 FakeGPS

主流模拟器无需配置开发者选项, 安装 FakeGPS 后直接使用。

  1. 在模拟器中打开 FakeGPS, 授予定位权限
  2. 调整速度 (建议 2.5 - 4.0 m/s)、圈数
  3. 点击 "开始"
  4. 在模拟器中打开运动 App (Keep、小米运动健康等), 开始记录

注意事项

  • 如定位未恢复, 关闭 FakeGPS 或重启模拟器即可
  • 速度不宜超过 5 m/s, 否则可能被判定异常
  • 模拟器支持 Keep、小米运动健康、华为运动健康等所有主流 App
